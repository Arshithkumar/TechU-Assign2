# TechU Assignment 2 
#### JavaScript API Integration and DOM Manipulation

A brief description of what this project does and who it's for

This assignment aims to assess your understanding of JavaScript, API integration, and
DOM manipulation. You will be required to create a simple web application that fetches
data from a public API and dynamically updates the DOM based on the retrieved data.
